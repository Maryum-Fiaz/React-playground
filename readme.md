## It's REACT PlayGround 🎮

*When I learn about a new concept, I play with that here!*

This project is a ready-made sandbox created with Vite and Tailwind CSS. I designed it to make practicing new concepts simple without wasting time setting up a new project from scratch.

---

## How to Contribute

Want to add your own small project or practice a concept? Follow these quick steps:

1. **Fork** this repository.
2. **Code your component** (see the Project Structure guide below).
3. **Add a new button** to the array.
4. Open a **Pull Request**!

---

## 📂 Project Structure & Step-by-Step Guide

To properly plug your mini-project into this playground, you only need to touch three places:

### 1. `smallProjects` folder
* Create your new project folder inside here.
* Write all your custom components, logic, and styles inside your folder.

### 2. `components` folder -> `Parent.jsx` file
* Open `Parent.jsx`.
* Add a new button object to the existing array of buttons. 
* Make sure to provide its label and URL path address

### 3. `main.jsx` file
* Open `main.jsx`.
* Import your main component from your `smallProjects` folder.
* Add a new `<Route>` mapping your specific URL path to your component.
